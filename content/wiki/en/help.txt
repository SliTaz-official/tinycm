<h2>Help</h2>

<p>
TinyCM let you easely create nice web content. Clean, fast and CGI SHell powered!
TinyCM provide a simple Wiki syntax, JavaScript code helper, diff interface, a dashboard, Hg integration and a plugin system. It can be also multi-user and a greate collaboration tool.
</p>

<h3>Titles</h3>
<pre>
==== Title 1 ====
=== Title 2 ===
== title 3 ==
</pre>

<h3>Text formating</h3>
<p>
This how to make <strong>bold</strong>, <em>emphasis</em> and <u>underline</u> text:
</p>
<pre>
**bold** ''emphasis'' __underline__
</pre>

<h3>Page link and URL</h3>
<p>
To create a new page you have to create a new link to it. Then you can edit and add content to your new page. First the link title then the path to a new or an existing page:
</p>
<pre>
[Help|en/help]
</pre>
<p>
You can also copy/past a full URL such as:
</p>
<pre>
http://www.slitaz.org/
</pre>

<h3>Images</h3>
<p>
You can have some images in your content folder or use on from the web. Just paste the URL to the PNG image:
</p>
<pre>
http://site.com/path/image.png
</pre>
